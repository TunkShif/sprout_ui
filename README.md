SproutUI provides unstyled and accessible Phoenix (Live View) components for building web apps.

## Installation

The package can be installed by adding `sprout_ui` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:sprout_ui, "~> 0.1.0"}
  ]
end
```
